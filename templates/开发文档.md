### 4.2 功能详细设计

#### 功能1：用户注册

**流程**：
```
1. 用户输入邮箱、密码
2. 前端验证输入格式
3. 调用 Supabase Auth API
4. 返回注册结果
5. 发送确认邮件
```

**接口**：
- `POST /auth/v1/signup`

**数据结构**：
```dart
class RegisterRequest {
  String email;
  String password;
}
```

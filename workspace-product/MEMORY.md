## 三、PRD 规范

### 3.1 PRD 模板

使用 `skills/PRD-TEMPLATE.md` 作为标准模板。

### 3.2 PRD 输出位置

```
~/.openclaw/projects/<项目名>/docs/PRD.md
```

### 3.3 PRD 质量标准

| 检查项 | 要求 |
|--------|------|
| 用户故事 | 至少 3 个，格式：As a... I want... So that... |
| 功能列表 | 按优先级 P0/P1/P2 分类 |
| 验收标准 | 每个功能必须有可测试的验收条件 |
| 数据模型 | 必须包含 ER 图或 SQL 设计 |
| API 设计 | 必须包含 RESTful 接口定义 |

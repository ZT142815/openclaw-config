## 五、测试执行

### 5.1 接口测试命令

```bash
# 测试 Supabase 连接
curl -X GET 'https://<project>.supabase.co/rest/v1/' \
  -H 'apikey: <ANON_KEY>'

# 测试用户注册
curl -X POST 'https://<project>.supabase.co/auth/v1/signup' \
  -H 'apikey: <ANON_KEY>' \
  -H 'Content-Type: application/json' \
  -d '{"email": "test@example.com", "password": "Test123456"}'
```

### 5.2 页面测试命令

```bash
# 启动 Appium
appium &

# 运行 Flutter 集成测试
flutter test integration_test/app_test.dart
```

## 六、缺陷管理

### 6.1 缺陷分级

| 级别 | 说明 | 处理方式 |
|------|------|----------|
| Critical | 核心功能完全不可用 | 立即修复 |
| High | 主要功能无法正常使用 | 优先修复 |
| Medium | 功能有问题但有替代方案 | 计划修复 |
| Low | 界面样式、文字等小问题 | 后续修复 |

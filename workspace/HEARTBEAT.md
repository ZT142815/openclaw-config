## ⚠️ 强制执行：每次回复必须写入记忆

> ⚠️ **不要跳过！每次回复用户前必须执行！**

---

### 固定流程（必须执行）

**每次回复前**：
```
1. 检查 memory/今日.md 是否存在
   ├─ 不存在 → 创建
   └─ 存在 → 直接追加

2. 追加本次对话到 memory/今日.md
```

---

### 追加格式（固定，直接复制使用）

```
## HH:MM

### 用户
[用户说的内容]

### AI
[AI回复的内容]
```

---

### 示例

```
## 10:20

### 用户
你觉得完善了吗？

### AI
记忆系统的定义层面已完善，但执行层面还有问题。

### 写入
✅ 已追加到 memory/2026-04-01.md
```

---

### 简化原则

| 之前（复杂） | 现在（简单） |
|-------------|-------------|
| 需要智能判断写入什么 | 直接复制对话内容 |
| 需要判断是否需要写入 | 每次都要写入 |
| 需要思考写入哪个文件 | 固定 memory/今日.md |

**成本：从"思考"变成"复制粘贴"**

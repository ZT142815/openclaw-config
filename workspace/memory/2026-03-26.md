## 团队基建项目：智能运营中心 Dashboard

### 项目路径
`~/.openclaw/projects/dashboard/`

### 技术栈
- **后端**: Python + FastAPI + WebSocket
- **前端**: React + TypeScript + Tailwind CSS + Framer Motion
- **设计风格**: 科技感深色 + 霓虹光效

### 已实现功能
1. Agent 状态监控（空闲/工作中/暂停）
2. 今日概览统计
3. 活动时间线（含详细描述、标注是谁）
4. 项目进度追踪（自动更新）
5. 本周趋势统计
6. 通知面板
7. 设置面板
8. 统计面板

### Dashboard 服务
- 后端: http://localhost:8080
- 前端: http://localhost:5173/5174 (注意：端口可能变化)

### 数据文件
```
~/.openclaw/projects/dashboard/backend/data/
├── agents.json      # Agent 状态
├── projects.json    # 项目列表
├── activities.json  # 活动记录
└── settings.json    # 设置
```

### API 接口
| 接口 | 方法 | 说明 |
|------|------|------|
| /api/agents | GET | 获取所有 Agent |
| /api/agents/{id}/status | PUT | 更新 Agent 状态（自动更新项目进度） |
| /api/projects | GET/POST | 项目列表/创建 |
| /api/activities | GET | 活动记录 |
| /api/stats/today | GET | 今日统计 |
| /api/stats/weekly | GET | 本周趋势 |

### 活动时间线详细描述
- 开始任务: "👨‍💻 Developer Agent 开始任务：正在「任务名称」（项目名称）📊 10%"
- 完成任务: "✅ Developer Agent 完成任务：「任务名称」已完成 📊 项目进度: 40%"

### 项目进度自动更新机制
- Agent 开始任务 → 自动创建/更新项目 → 进度 = 10%
- Agent 完成任务 → 自动增加进度 → 进度 += 30%

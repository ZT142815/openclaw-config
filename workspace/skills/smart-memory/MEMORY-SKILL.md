# 🧠 智能记忆系统

> **作用**：管理 Agent 的记忆读取、写入和整理
> **适用**：所有 Agent（CEO / Developer / Product / Tester）
> **版本**：v3.2
> **更新日期**：2026-04-01

---

## 一、核心原则

```
❌ 过去的问题：
- 问"今天干了什么" → 不知道，靠猜
- 记忆不准确、不完整

✅ 现在的目标：
- 问"今天干了什么" → 读取 memory/今日.md → 准确回答
- 用户说了什么、Agent回复了什么、Agent干了什么 全部记录
```

## 三、启动读取流程

> ⚠️ **每次 session 启动时必须执行**

### 第一步：动态日期计算

```python
当前日期 = "2026-04-01"
今日文件名 = f"memory/{当前日期}.md"  # memory/2026-04-01.md
昨日日期 = 当前日期 - 1天  # 2026-03-31
昨日文件名 = f"memory/{昨日日期}.md"  # memory/2026-03-31.md
```

### 第二步：初始化今日记忆文件

```python
今日文件 = f"{工作空间}/memory/{今日日期}.md"

if not exists(今日文件):
    创建空文件(今日文件)
    写入文件头(今日文件, 今日日期)
else:
    直接读取
```

**文件头格式**：
```markdown
# YYYY-MM-DD 工作记录

> 创建时间: YYYY-MM-DD HH:MM
```

### 第三步：读取以下文件（按顺序）

```
1. IDENTITY.md      → 身份定位
2. USER.md          → 用户信息
3. MEMORY.md        → 长期记忆（项目、决策、团队）
4. memory/昨日.md   → 昨天干了什么
5. memory/今日.md   → 今天干了什么
6. AGENTS.md        → 团队成员和调度命令
7. TOOLS.md         → 可用工具配置
8. RULES.md         → 团队规范
```

**文件读取规则**：
```
for 文件 in 文件列表:
    if exists(文件):
        try:
            读取文件内容
        except 读取错误:
            记录错误，继续下一个
            print(f"⚠️ {文件} 读取失败，跳过")
    else:
        print(f"⏭️ {文件} 不存在，跳过")
        continue
```

### 第四步：Mem0 向量搜索

```python
# 根据不同Agent类型，使用不同的查询词
queries = {
    "CEO": [
        "当前项目进展和任务状态",

    ],
    "Developer": [
        "项目开发进度",

        "竞品分析结论"
    ],
    "Tester": [
        "测试进度和BUG状态",
        "测试用例和结果"
    ]
}

for query in queries[当前Agent类型]:
    results = memory_search(query=query, scope="long-term", limit=3)
    整合到上下文
```

**Mem0 结果处理**：
```
如果找到相关记忆：
    → 将结果整合到当前上下文
    → 用于补充今日工作的背景信息

如果未找到：
    → 继续，不影响主流程
```

### 第五步：读取失败处理

```python
读取结果 = {
    "成功": [],

for 文件 in 文件列表:
```

## 四、智能写入流程

> ⚠️ **对话过程中实时执行，边对话边写入**

### 触发时机

```
┌─────────────────────────────────────────────────────────┐
│                    对话进行中                            │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  用户发送消息 ──→ 分析内容 ──→ 判断写入 ──→ 执行写入     │
│       ↑                              ↓                   │
│       └──────── 继续对话 ←────── 写入完成                 │
│                                                          │
└─────────────────────────────────────────────────────────┘

触发点：
1. 用户发送消息后 → 分析并写入
2. Agent 回复前 → 检查是否需要写入
3. Agent 执行操作后 → 记录操作
4. 每次关键操作完成 → 立即写入
```

### 写入来源

| 来源 | 说明 | 记录位置 |
|------|------|----------|
| **用户说了什么** | 需求、偏好、决策、反馈 | USER.md / MEMORY.md / memory/今日 |
| **Agent 回复了什么** | 分析、方案、确认、汇报 | memory/今日 |
| **Agent 干了什么** | 执行的操作、派发的任务、修改的文件 | memory/今日 |

### 智能判断算法（v3.0）

> **核心思想**：不是所有对话都需要写入，只写入"有意义的事"

#### 判断流程

```
用户消息 → 提取关键词 → 判断类型 → 决定写入目标
```

#### 判断规则

**🔴 必须写入（操作/决策类）**

```python
# 1. 操作执行
if 包含任意(["删除", "更新", "修改", "配置", "调整", "新增", "创建"]):
    写入 memory/今日.md
    格式：### [HH:MM] [操作用户] 执行了 [操作内容] → [结果]

# 2. 决策确认
if 包含任意(["好的", "就这样", "开始吧", "执行", "确认"]):
    if 上下文 in ["方案讨论", "操作确认", "任务派发"]:
        写入 memory/今日.md
        格式：### [HH:MM] [用户] 确认 [决策内容]

# 3. 问题解决
if 包含任意(["解决了", "修复了", "排查到", "找到原因"]):
    写入 memory/今日.md
    格式：### [HH:MM] 问题解决：[问题] → [解决方案]
```

**🟡 应该写入（分析/汇报类）**

```python
# 4. 重要分析
if 包含任意(["发现问题", "找到问题", "分析结果", "排查"]):
    写入 memory/今日.md

# 5. 进度汇报
if 包含任意(["完成了", "进度", "状态"]):
    if 涉及任务或项目:
        写入 memory/今日.md
```

**🟢 不写入（闲聊/询问类）**

```python
# 6. 身份询问
if 匹配(r"^(你是谁|你叫什么|你是干什么的)"):
    不写入

# 7. 问候寒暄
if 匹配(r"^(你好|您好|谢谢|辛苦了|再见)"):
    不写入

# 8. 简单确认
if 匹配(r"^(好的|好的吧|行)"):
    if 字数 < 10:
        不写入

# 9. 纯询问
if 包含任意(["现在几点|今天几号|帮我看看"]):
    if 不涉及操作:
        不写入
```

#### v3.2 智能判断算法（改进版）

```python
class 智能记忆系统:
    """v3.2 智能记忆系统"""
    
    def __init__(self):
        # 上下文：保存上一轮对话
        self.上下文 = {
            "上一轮用户": None,
            "上一轮AI": None,
            "待确认操作": None  # 用户刚确认的操作
        }
    
    def 智能判断(self, 用户消息, Agent操作=None):
        """
        v3.2 智能判断：
        1. 操作类优先（最高）
        2. 决策类次之
        3. 偏好类再次
        4. 闲聊类过滤（最低）
        5. 支持多文件写入
        6. 支持上下文感知
        7. 自动记录Agent操作
        """
        
        结果 = {"写入": [], "不写入": False, "Agent操作": []}
        消息 = 清理(用户消息)
        
        # =====================
        # 0. Agent操作自动记录（最高优先级）
        # =====================
        if Agent操作:
            结果["Agent操作"].append(Agent操作)
            结果["写入"].append("memory/今日.md")
        
        # =====================
        # 1. 第一步：提取所有匹配的文件（可能多个）
        # =====================
        匹配的文件 = self._提取匹配文件(消息)
        
        # =====================
        # 2. 第二步：结合上下文调整
        # =====================
        匹配的文件 = self._上下文调整(消息, 匹配的文件)
        
        # =====================
        # 3. 第三步：判断是否不写入
        # =====================
        if self._是闲聊类(消息, 匹配的文件):
            结果["写入"] = []
            结果["不写入"] = True
            return 结果
        
        # =====================
        # 4. 第四步：优先级排序并返回
        # =====================
        结果["写入"] = 匹配的文件
        return 结果
    
    def _提取匹配文件(self, 消息):
        """提取所有匹配的文件"""
        文件列表 = []
        
        # USER.md - 偏好/家人
        if 包含任意(消息, ["喜欢", "讨厌", "想要", "希望", "偏好", "习惯", "老婆", "女儿", "家人"]):
            文件列表.append("USER.md")
        
        # RULES.md - 规范
        if 包含任意(消息, ["以后都", "规则是", "流程是", "原则"]):
            文件列表.append("RULES.md")
        
        # AGENTS.md - 团队变更
        if 包含任意(消息, ["新增成员", "调整分工", "团队变更", "成员变更"]):
            文件列表.append("AGENTS.md")
        
        # MEMORY.md - 决策/项目
        if 包含任意(消息, ["决定", "就这样", "选择", "用这个", "不用这个", "确认", "新建项目", "项目完成"]):
            文件列表.append("MEMORY.md")
        
        # PROJECT.md - 项目管理
        if 包含任意(消息, ["项目管理", "项目规范"]):
            文件列表.append("PROJECT.md")
        
        # memory/今日.md - 操作/完成（最高优先级）
        if 包含任意(消息, ["删", "更新", "修改", "配置", "新增", "创建", "完成", "解决", "排查"]):
            if "memory/今日.md" not in 文件列表:
                文件列表.insert(0, "memory/今日.md")  # 插入到最前面
        
        # IDENTITY.md - 身份修改
        if 包含任意(消息, ["身份要改", "性格要改", "语气要改", "名字要改"]):
            文件列表.append("IDENTITY.md")
        
        # SOUL.md - 启动流程
        if 包含任意(消息, ["启动流程", "周小墨能力", "职责要改"]):
            文件列表.append("SOUL.md")
        
        # TOOLS.md - 工具配置
        if 包含任意(消息, ["工具配置", "新增工具", "配置工具"]):
            文件列表.append("TOOLS.md")
        
        return 文件列表
    
    def _上下文调整(self, 消息, 文件列表):
        """结合上下文调整判断"""
        
        # 场景1：用户说"好的"确认上轮AI的操作
        if 消息 in ["好的", "好的吧", "行", "执行", "开始"]:
            if self.上下文["待确认操作"]:
                if "memory/今日.md" not in 文件列表:
                    文件列表.insert(0, "memory/今日.md")
                return 文件列表
        
        # 场景2：用户说"帮我XXX"，这是操作请求
        if 消息.startswith("帮我") or 消息.startswith("帮我把"):
            if "memory/今日.md" not in 文件列表:
                文件列表.insert(0, "memory/今日.md")
        
        return 文件列表
    
    def _是闲聊类(self, 消息, 文件列表):
        """判断是否闲聊类"""
        
        # 纯问候/询问
        if 匹配开头(消息, ["你是谁", "你好", "您好", "谢谢", "辛苦了", "现在几点", "今天几号"]):
            return True
        
        # 纯短回复（<5字）且没有匹配任何文件
        if 字数(消息) < 5 and len(文件列表) == 0:
            return True
        
        return False
    
    def 更新上下文(self, 用户消息, AI回复, 操作=None):
        """更新上下文"""
        self.上下文["上一轮用户"] = 用户消息
        self.上下文["上一轮AI"] = AI回复
        self.上下文["待确认操作"] = 操作
```

### 主语识别规则

```python
def 识别主语(文本):
    # 用户说的（第一人称）
    if 匹配(r"^我", 文本):
        return "用户"
    
    # 用户说的（第二人称）
    if 匹配(r"^你", 文本):
        return "Agent"
    
    # Agent说的
    if 匹配(r"^✅|^💾|^📖|^🔍", 文本):
        return "Agent"
    
    # 无主语，根据上下文判断
    return "上下文推断"
```

### 内存缓冲机制

```python
# 使用内存缓冲，减少磁盘IO

写入缓冲 = {
    "memory/今日.md": [],
    "USER.md": [],
    "MEMORY.md": []

def 添加到缓冲(目标文件, 内容):
    写入缓冲[目标文件].append({

    })

def 刷新缓冲():

    for 目标文件, 记录列表 in 写入缓冲.items():
        if 记录列表:
            for 记录 in 记录列表:

            清空(记录列表)
```

### 去重检查

```python
def 需要写入(目标文件, 新内容):
    """检查是否需要写入"""
    
    # 读取目标文件
    if not exists(目标文件):
        return True  # 文件不存在，需要创建
    
    现有内容 = 读取文件(目标文件)
    
    # 完全相同检查
    if 新内容 in 现有内容:
        print(f"⏭️ 内容已存在，跳过: {新内容[:50]}...")
        return False
    
    # 相似度检查（关键词匹配）
    新内容关键词 = 提取关键词(新内容)
    for 行 in 现有内容.split("\n"):
        行关键词 = 提取关键词(行)
        相似度 = 计算相似度(新内容关键词, 行关键词)
        if 相似度 > 0.8:
            print(f"⏭️ 内容相似度{相似度*100:.0f}%，跳过: {新内容[:50]}...")
            return False
    
    return True  # 新内容，需要写入
```
